### **RealTime Chat App Using Flask and SocketIO**


**Dependancies:**

- Flask 
- Flask-SocketIO 
- Flask-Session 
- Eventlet 

To install run the followinf command:
```python
pip install flask Flask-SocketIO Flask-Session eventlet 
```
